Sales System




Tech Used:
  • Visual Studio 2010
  • Access Database
  • C#.NET

We developed a Sales System that could be dynamic and useable for many business situations. The system logs everything from sales, products, employees, customers, shippers and suppliers all into a local access database. Doing so gives this system more flexibility for changes that are guaranteed to happen in the future.

We used the MVC pattern in this system because of the flexibility that it brings and for future maintainability.
